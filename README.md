1. [Install uv](https://docs.astral.sh/uv/getting-started/installation/)
2. Run `uv sync` in your terminal to install all dependencies.
3. Select the virtual environment that uv creates in the top-right corner of any `.ipynb` notebook you wish to run in the environment that has the dependencies installed.
